Ten Hundred Words

So how does this thing work?

This editor is obviously inspired by XKCD's Upgoer Five rocket diagram. I didn't love the existing editor I found, which is great but not quite what I wanted. I wanted to make something very visual, that used the same wordset Randall Monroe used for XKCD.

I took the Wiktionary Contemporary Fiction Frequency List, and wrote a quick PHP script to run it through the Automatically Generated Inflection Database (part of the SCOWL family of word things).

This gave me a long list of words and their variants, plurals, etc, which seems very close to the "canonical" Ten Hundred Word list used for XKCD.

I wrote a simple jQuery script to compare incoming input to an array of good words, hand-crafted some HTML5 and CSS (using SASS and Bourbon), and here we are.
